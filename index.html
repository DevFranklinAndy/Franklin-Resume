Who Am I

I am a MERN Stack Developer who enjoys solving real problems through thoughtful software implementations. I build end-to-end web applications using Node.js, Express.js and MongoDB (Mongoose) on the backend, where I focus on data modeling, API design, and system reliability. On the frontend, I work with React, Next.js, TailwindCSS, and SCSS to create interfaces that are responsive, intuitive, and built with real users in mind. I use Git and GitHub to manage codebases and collaborate effectively, and I pay close attention to performance, maintainability, and long-term impact. My goal is not just to build features, but to deliver solutions that are practical, scalable, and meaningful.
